# GitTrace

GitTrace is a Chrome Extension that analyzes GitHub repositories to detect AI-generated code patterns, identify license risks, surface security vulnerabilities, and evaluate project compatibility.

---

## Project Structure

```text
gittrace/
├── extension/     # Chrome Extension (Manifest V3)
├── backend/       # Node.js / Express API
└── shared/        # Shared constants, utilities, and types
```
