# GitTrace

Chrome extension that analyzes GitHub repositories for AI-generated code,
license risk, security vulnerabilities, and system compatibility.

## Structure

```
gittrace/
├── extension/     # Chrome Extension (Manifest V3)
├── backend/       # Node.js / Express API
└── shared/        # Shared constants and types
```

## Stack

- Extension: Vanilla JS, Chrome MV3, Shadow DOM
- Backend: Node.js, Express, Axios
